# Certain full armor sets grant infinity potion effects - E.g. when a player wears a ruby helmet, ruby chestplate, ruby leggings and ruby boots, the player will be granted fire resistance. When the player removes one of the armor pieces, the potion effect will instantly disappear. Below is a list of all the armors that can be created by mOres. If the armor has a bonus it will be described below. ## Chainmail - No bonus effect for full set *** ## Copper - No bonus effect for full set *** ## Tin - Jump boost potion effect for full set *** ## Bronze - No bonus effect for full set *** ## Silver - No bonus effect for full set *** ## Sterling Silver - No bonus effect for full set *** ## Cobalt - Haste bonus effect for full set *** ## Carbon Steel - No bonus effect for full set *** ## Amethyst - Slow falling bonus effect for full set *** ## Obsidian - Strength & slowness bonus effect for full set *** ## Emerald - Luck bonus effect for full set *** ## Topaz - Haste 2 bonus effect for full set *** ## Tourmaline - Regeneration bonus effect for full set *** ## Tanzanite - Speed bonus effect for full set *** ## Ruby - Fire resistance bonus effect for full set *** ## Sapphire - Night vision bonus effect for full set *** ## Moissanite - No bonus effect for full set *** ## Carbonado - No bonus effect for full set *** ## Turquoise - Water breathing & dolphin effect for full set *** ## Graphene - No effect for full set ***
